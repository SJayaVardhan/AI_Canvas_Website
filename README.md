🎨 AI Canvas

 Draw your questions, get instant AI answers!

An intelligent drawing canvas that recognizes handwritten text and math problems, then provides AI-powered responses in real-time.

## ✨ Features

- **🖊️ Smart Drawing Canvas** - Natural drawing experience with multiple pen colors and sizes
- **🧠 AI-Powered Recognition** - Advanced handwriting recognition using multiple AI services
- **⚡ Instant Answers** - Get immediate responses to questions, math problems, and more
- **🎯 Multiple Recognition Methods** - Gemini Vision, Groq AI, OCR, and smart pattern recognition
- **📱 Mobile Friendly** - Works perfectly on phones, tablets, and desktops
- **🚀 No Installation** - Pure web-based, works in any modern browser

## 🎯 How to Use

1. **Draw** your question or math problem on the canvas
2. **Click "✨ Analyze"** to process your drawing
3. **Get instant answers** powered by AI
4. **Continue drawing** for more questions
